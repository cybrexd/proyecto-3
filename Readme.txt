El dashboard de power bi recoge datos de la api sobre el mercado de criptomonedas y asi representarlos estableciendo relaciones
entre los mismos.

Los datos de las cryptos seleccionadas obtenidos atravez de la api fueron los siguientes:

Fecha de registro(diario) (Permitiendo conocer a lo largo del tiempo el precio y el volumen que manejo esa criptomoneda, pudiendo expresarlo de manera grafica)
Precio mas alto y Precio mas bajo (Permite Conocer las variaciones de precio del dia a dia)
Precio de cierre (Es el precio final y de referencia siendo el dato mas importante obtenido, debido a que cada registro de este permite analizar la fluctacion y evolucion del precio de la cripto)
Volumen de Transaccion (El saber el Volumen de transaccion nos ayuda a mostrar cuanto dinero es transaccionado con esa cripto, pudiendonos dar una idea de que tan grande y tan utilizada es la moneda)
